# ✈️ Airline Price vs Oil Price (Under Construction)

## 📌 Overview

This project explores the relationship between airline ticket prices and crude oil prices. Since fuel cost is a major operational expense for airlines, fluctuations in oil prices can directly impact ticket pricing.

The goal of this project is to:

* Analyze historical trends between airline fares and oil prices
* Perform data preprocessing and feature engineering
* Build predictive models to understand correlations and forecasting

⚠️ **Note:** This project is currently under development. Features and results may change.

---

## 🧠 Problem Statement

Airline ticket prices are influenced by multiple factors such as demand, seasonality, and fuel costs. Among these, oil prices play a crucial role.

This project attempts to answer:

* Is there a measurable correlation between oil prices and airline ticket prices?
* Can we predict airline ticket prices using oil price trends?

## ⚙️ Installation & Setup

### 🔹 Option 1: Run Locally

```bash
git clone https://github.com/Vineet-Kumar-001/Airline-price-vs-oil-price-under-construction-.git
cd Airline-price-vs-oil-price-under-construction-
pip install -r requirements.txt
```

---

### 🔹 Option 2: Run using Docker (Recommended)

Pull the prebuilt Docker image:

```bash
docker pull vineetkumar001/airline_price_vs_oil_price:latest
```

Run the container:

```bash
docker run -p 8888:8888 vineetkumar001/airline_price_vs_oil_price:latest
```

Then open your browser and go to:

```
http://localhost:8888
```
